wmc2d

About

wmc2d is a dockapp to monitor the core 2 duo temperature and frequency.

License AGPLv3: GNU Affero General Public License version 3

This is a small dockapp, which shows the core 2 duo cpu temperatures and the temperature of ACPI thermal zone 0, which is normaly the motherboard temperature.

Only libxcb (X C-language Bindings library) and the standard C library are as external libraries required.
